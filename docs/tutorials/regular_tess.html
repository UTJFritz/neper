Generating a Regular Tessellation

Regular tessellations, for which all cells have the same shape, can be generated. Available morphologies are squares in 2D, and cubes and truncated octahedra in 3D.

Note

To reproduce exactly the images below, add the following line to your $HOME/.neperrc file (or to a local configuration file to be loaded with --rcfile):

neper -V -imagesize 800:400

Generating a Square-Cell Tessellation

A tessellation composed of \(4 \times 4 \times 4\) squares can be generated as follows:

$ neper -T -n from_morpho -morpho "square(4)" -id 2 -dim 2 -o square4

This produces a Tessellation File (.tess) named square4.tess.

The tessellation can be visualized using the Visualization Module (-V):

$ neper -V square4.tess -print img1
